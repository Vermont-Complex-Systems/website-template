# Scrollytelling Template (Minimal)

A clean starting point for scrollytelling projects.

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:5173 to see your starter story.

## Next Steps

1. **Edit content**: `src/lib/stories/my-first-story/data/copy.json`
2. **Add visualization**: `src/lib/stories/my-first-story/components/Index.svelte`
3. **Customize styles**: `src/styles/app.css`
4. **Add more stories**: Create new folders in `src/lib/stories/`

## Structure

```
src/
├── lib/stories/my-first-story/   # Your starter story
├── data/stories.csv              # Story routing
└── styles/app.css                # Your customizations
```

## Build & Deploy

```bash
npm run build
```

Deploy the `build/` folder to GitHub Pages, Vercel, or Netlify.
